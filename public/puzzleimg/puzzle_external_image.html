PuzzleImg example - Puzzle with external Image

Here it is added the canvas with the puzzle.
- Puzzle game with an image from external location (or an URL address), created with PuzzleImg.
- Code: 
<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8" />
<title>PuzzleImg example - Puzzle with external Image</title>
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="puzzleimg.css" />
<script src="puzzleimg.js"></script>
</head>
<body>
<div id="puz1" class="puzelm">Here it is added the canvas with the puzzle.</div>
<script>
//<![CDATA[
//Creates a puzzle in the #puz1 element with an image from an external location
//Having 5 columns and 4 rows. Setting canvas size: width=600 and height=450
var puz1 = new PuzzleImg('puz1', 'waterfall.jpg', 5, 4, 600, 450);

//rewrites the function that is called when the puzzle is completed
//the "clicks" property contains the number of clicks
puz1.solved = function(){
  alert('Congratulations, you solved the puzzle in '+ puz1.clicks +' clicks');
}
// ]]>
</script>
</body>
</html>
